# Tools Service

Free email verification tools - no authentication or credits required.

## Quick Start

| Property | Value |
|----------|-------|
| **Location** | `apps/backend/tools` |
| **Port** | 8006 |
| **Start Command** | `bun run be:tools:dev` |

## Environment Variables

```bash
PORT=8006
BASE_URL=https://local.verifio.email
REDIS_URL=redis://:verifio123@localhost:6379
RATE_LIMIT_MAX_REQUESTS=100
RATE_LIMIT_WINDOW_MS=60000
```

## Main Endpoints

| Route | Method | Description | Auth Required |
|-------|--------|-------------|---------------|
| `/api/tools/v1/syntax/validate` | POST | Validate email syntax | No |
| `/api/tools/v1/disposable/check` | POST | Check disposable email | No |
| `/api/tools/v1/deliverability/check` | POST | Basic deliverability | No |
| `/api/tools/v1/list-health` | GET | All service health | No |
| `/api/tools/health` | GET | Health check | No |

## Using the Tools

**Syntax Validation:**
```bash
curl -X POST https://api.verifio.email/api/tools/v1/syntax/validate \
  -H "Content-Type: application/json" \
  -d '{"email": "user@example.com"}'
```

**Disposable Check:**
```bash
curl -X POST https://api.verifio.email/api/tools/v1/disposable/check \
  -H "Content-Type: application/json" \
  -d '{"email": "user@tempmail.com"}'
```

**Deliverability Check:**
```bash
curl -X POST https://api.verifio.email/api/tools/v1/deliverability/check \
  -H "Content-Type: application/json" \
  -d '{"email": "user@example.com"}'
```

## Rate Limiting

- **Per-IP limits** to prevent abuse
- **Redis-backed** for distributed systems
- **Default:** 100 requests per 60 seconds

## Features

- **No authentication** required
- **No credits** needed
- **Rate limited** per IP
- **Basic checks only** (no SMTP handshake)

## Common Issues

**"Rate limit exceeded":**
- Wait for time window to reset (default 60 seconds)
- Reduce request frequency

**Redis connection errors:**
- Verify Redis is running
- Check `REDIS_URL` in `.env`

# Auth Service

Central authentication handling user accounts, sessions, OAuth, and organizations.

## Quick Start

| Property | Value |
|----------|-------|
| **Location** | `apps/backend/auth` |
| **Port** | 8000 |
| **Start Command** | `bun run be:auth:dev` |

## Environment Variables

```bash
PORT=8000
BASE_URL=https://local.verifio.email
PG_URL=postgresql://verifio:verifio123@localhost:5432/verifio
REDIS_URL=redis://:verifio123@localhost:6379
BETTER_AUTH_SECRET=<generate with: openssl rand -hex 32>
ENCRYPTION_KEY=<generate with: openssl rand -hex 32>
```

**Optional (OAuth):**
```bash
GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=
GITHUB_CLIENT_ID=
GITHUB_CLIENT_SECRET=
```

## Main Endpoints

| Route | Method | Description |
|-------|--------|-------------|
| `/api/auth/v1/sign-up` | POST | Create account |
| `/api/auth/v1/sign-in/email` | POST | Login with email/password |
| `/api/auth/v1/sign-out` | POST | Logout |
| `/api/auth/v1/session` | GET | Get current session |
| `/api/auth/health` | GET | Health check |

## Database Tables

- `user` - User accounts
- `session` - Active sessions
- `organization` - Organizations
- `member` - Organization memberships
- `account` - OAuth connections

## Common Issues

**Service won't start:**
- Check port 8000 is available
- Verify `BETTER_AUTH_SECRET` is set

**OAuth not working:**
- Verify OAuth credentials are set
- Check callback URL: `https://local.verifio.email/api/auth/callback/*`

**Session issues:**
- Verify Redis is running
- Check session cookie is being sent

Core email verification with syntax, DNS, MX, and SMTP checks.

## Quick Start

| Property | Value |
|----------|-------|
| **Location** | `apps/backend/verify` |
| **Port** | 8001 |
| **Start Command** | `bun run be:verify:dev` |

## Environment Variables

```bash
PORT=8001
BASE_URL=https://local.verifio.email
PG_URL=postgresql://verifio:verifio123@localhost:5432/verifio
REDIS_URL=redis://:verifio123@localhost:6379
MAX_BULK_EMAILS=10000
DEFAULT_TIMEOUT=5000
```

## Main Endpoints

| Route | Method | Description | Auth Required |
|-------|--------|-------------|---------------|
| `/api/verify/v1/verify` | POST | Verify single email | API Key or Session |
| `/api/verify/v1/bulk-verify` | POST | Start bulk job | API Key or Session |
| `/api/verify/v1/bulk-jobs/:jobId` | GET | Get job status | API Key or Session |
| `/api/verify/v1/bulk-jobs/:jobId/results` | GET | Get job results | API Key or Session |
| `/api/verify/v1/history` | GET | Get verification history | API Key or Session |
| `/api/verify/health` | GET | Health check | No |

## Authentication

**API Key:**
```bash
curl -H "Authorization: Bearer rl_your_key" \
  https://api.verifio.email/api/verify/v1/verify \
  -d '{"email": "user@example.com"}'
```

**Session Cookie:**
```bash
curl --cookie-jar cookies.txt --cookie cookies.txt \
  https://api.verifio.email/api/verify/v1/verify \
  -d '{"email": "user@example.com"}'
```

## Dependencies

- **Auth Service** - Session validation
- **Credits Service** - Check/deduct credits
- **Logs Service** - Activity logging

## Bulk Verification

1. Check credits
2. Create job record
3. Process in batches of 10
4. Update progress
5. Deduct credits
6. Mark complete

## Database Tables

- `verificationJob` - Bulk job tracking
- `verificationResult` - Individual results
- `apikey` - API key validation

## Common Issues

**"Insufficient credits":**
- Check Credits service is running
- Verify organization has credits available

**Job stuck at "processing":**
- Verify Credits service is running
- Check service logs

**Verification timeout:**
- Increase `DEFAULT_TIMEOUT`
- Check network connectivity

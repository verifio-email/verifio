# Upload Service

Handles file uploads for bulk email verification jobs.

## Overview

| Property | Value |
|----------|-------|
| **Location** | `apps/backend/upload` |
| **Framework** | ElysiaJS |
| **Port** | 8018 |
| **API Route** | `/api/upload/*` |

## Starting the Service

```sh
bun run be:upload:dev
```

## Features

- **CSV upload** - Accept email list CSV files
- **File validation** - Verify file format and size
- **Email extraction** - Parse emails from uploaded files
- **Job creation** - Trigger bulk verification workflow
- **Progress tracking** - Monitor upload and processing status

## Supported Formats

| Format | Extensions |
|--------|------------|
| CSV | `.csv` |
| Text | `.txt` |

## Upload Flow

```
File Upload
    ↓
Validate Format
    ↓
Extract Emails
    ↓
Check Credits
    ↓
Create Bulk Job → Trigger Workflow
    ↓
Return Job ID
```

## Environment Variables

```sh
PORT=8018
PG_URL=postgresql://verifio:verifio123@localhost:5432/verifio
```

## Related Services

- **Workflow** - Triggered after successful upload
- **Credits** - Validates sufficient credits for job size

<Callout type="warn">
  Large files are processed asynchronously via the workflow service.
</Callout>

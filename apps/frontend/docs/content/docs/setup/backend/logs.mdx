# Logs Service

Stores and retrieves verification history for dashboard display and analytics.

## Overview

| Property | Value |
|----------|-------|
| **Location** | `apps/backend/logs` |
| **Framework** | ElysiaJS |
| **Port** | 8020 |
| **API Route** | `/api/logging/*` |

## Starting the Service

```sh
bun run be:logs:dev
```

## Features

- **Verification history** - Store all verification results
- **Filtering** - Query by date, status, domain
- **Pagination** - Efficient large dataset handling
- **Export** - Download verification logs as CSV
- **Retention** - Configurable log retention policies

## Log Entry Structure

```typescript
interface VerificationLog {
  id: string
  email: string
  state: 'deliverable' | 'undeliverable' | 'risky' | 'unknown'
  score: number
  checks: object
  createdAt: Date
  organizationId: string
  apiKeyId: string
}
```

## Environment Variables

```sh
PORT=8020
PG_URL=postgresql://verifio:verifio123@localhost:5432/verifio
```

## Database Tables

- `verification_log` - Individual verification results
- Indexed by organization, date, and email domain
